# Unit 04: UI Test Flows

## Goal

Teach students how to combine actions and validations into a small business flow.

## What To Teach

- read a scenario
- choose the important actions
- choose the important validations
- keep one clear purpose per test

## Files In This Unit

- `example.spec.js`
- `exercise.spec.js`
